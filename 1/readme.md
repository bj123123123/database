生成一个python文件fund.py，需求如下
定义一个类，全局变量1:存放基金代码的一个list；全局变量2:一个字典，key是基金代码，value是基金净值
类的初始化函数，初始化全局变量1和全局变量2。如果存在fundvalue.txt文件，清空fundvalue.txt文件。如果fund_tmp/下存在文件，删除fund_tmp/下所有文件。
第一个函数读取本地的文件fundlist.txt，每一行是一个基金代码，把不重复的存放到全局变量1中。
第二个函数，入参1是基金代码,抓取网页内容，网页的url是“https://fund.eastmoney.com/”+入参1+".html"，获得<div class="dataOfFund">和</div>之间的内容，称为内容1，
    如果失败，则打印失败的信息，格式如下：“基金代码：入参1，抓取失败！！！”
    如果成功，则从内容1中删除“<!--”和“-->”之间的内容，称为内容2，并把内容2生成一个xml格式的文件，文件名是入参1+".xml"，放到fund_tmp/下，
        在内容2中找到第一个<dd class="dataNums">和</dd>之间的内容，称为内容3，然后在内容3中把<span>和</span>之间的内容取出来，会有两个值，第一个是净值，第二个是百分比。
            如果净值是数字，则key是入参1，value是净值，放到全局变量2中。
            如果净值不是数字，则key是入参1，value是0，放到全局变量2中。         
第三个函数，循环遍历全局变量1，取出每一个基金代码，调用第二个函数。遍历完成后，把全局变量2写入到一个txt文件中，文件名是fundvalue.txt，格式如下：
key\tvalue
key\tvalue
...
第四个函数，打印全局变量2，格式如下：
key:value
key:value
...

main函数，生成这个类的实例，调用第一个函数，然后调用第三个函数。完成输出成功完成的提示
